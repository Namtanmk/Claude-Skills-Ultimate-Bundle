# เริ่มต้นใช้งาน — AI Flow

ยินดีต้อนรับสู่ AI Flow คู่มือนี้จะช่วยให้คุณเริ่มใช้งานได้ภายในไม่ถึง 2 นาที

---

## สิ่งที่รวมอยู่ใน Bundle นี้

คุณมี **ทักษะมากกว่า 500 รายการ** จัดเป็น 20 หมวดหมู่:

- AI และเทคโนโลยี
- โฆษณาและสื่อแบบชำระเงิน
- การวิเคราะห์และข้อมูล
- การสร้างแบรนด์และการออกแบบ
- ลูกค้าและการให้คำปรึกษา
- เนื้อหาและการเขียนคอปปี้
- คอร์สและการศึกษา
- อีคอมเมิร์ซและผลิตภัณฑ์
- การตลาดอีเมลและระบบอัตโนมัติ
- งานอีเวนต์และการพูด
- การเงินและการตั้งราคา
- HR และทีมงาน
- เฉพาะอุตสาหกรรม
- การเปิดตัวและการเติบโต
- กฎหมายและการปฏิบัติตามกฎ
- องค์กรไม่แสวงหากำไรและชุมชน
- การดำเนินงานและระบบ
- SEO และการค้นหา
- การขายและ Funnel
- โซเชียลมีเดีย

แต่ละทักษะคือโฟลเดอร์ที่มีไฟล์ `SKILL.md` — ชุดคำสั่งที่มีโครงสร้างชัดเจน ซึ่งบอก Claude ว่าต้องจัดการงานธุรกิจเฉพาะนั้นอย่างไร

---

## การติดตั้ง

### Claude.ai (เว็บ)

1. เปิด [claude.ai](https://claude.ai) แล้วคลิกไอคอนโปรไฟล์
2. ไปที่ **Customize** → **Skills**
3. อัปโหลดโฟลเดอร์ทักษะที่คุณต้องการใช้
4. Claude จะตรวจจับและใช้งานโดยอัตโนมัติเมื่อเกี่ยวข้องกับการสนทนาของคุณ

### Claude Code (CLI)

คัดลอกโฟลเดอร์ทักษะไปยังไดเรกทอรีทักษะของ Claude Code:

**ทำให้ทักษะพร้อมใช้งานทุกที่ (แนะนำ):**

```bash
cp -r "AI Flow"/* ~/.claude/skills/
```

**ทำให้ทักษะพร้อมใช้งานเฉพาะในโปรเจกต์เดียว:**

```bash
cp -r "AI Flow"/* your-project/.claude/skills/
```

เพียงเท่านี้ ทักษะจะพร้อมใช้งานทันที

### Cowork

1. เปิด Cowork workspace ของคุณ
2. ไปที่การตั้งค่า Skills
3. อัปโหลดหรือซิงค์โฟลเดอร์ทักษะ
4. ทักษะจะทำงานอัตโนมัติตามคำขอของคุณ

---

## วิธีใช้งานทักษะ

### การตรวจจับอัตโนมัติ

เพียงอธิบายสิ่งที่คุณต้องการในภาษาธรรมดา Claude จะตรวจจับทักษะที่เหมาะสมและแนะนำคุณผ่านกระบวนการนั้น

**ตัวอย่าง:**
- "เขียนบล็อกโพสต์เกี่ยวกับเคล็ดลับการเพิ่มผลผลิต" → เรียกใช้ทักษะ `blog-post`
- "ฉันต้องการสัญญาสำหรับนักออกแบบฟรีแลนซ์" → เรียกใช้ทักษะ `contract-writer`
- "ช่วยวางแผนการเปิดตัวผลิตภัณฑ์" → เรียกใช้ทักษะ `launch-checklist`

### คำสั่ง Slash (Claude Code)

ใน Claude Code คุณสามารถเรียกใช้ทักษะใดก็ได้โดยตรงด้วยคำสั่ง slash:

```
/blog-post
/email-sequence
/sales-funnel-builder
/contract-writer
/sop-builder
```

ทักษะจะถามคำถามที่ถูกต้อง จากนั้นส่งมอบผลลัพธ์ที่สมบูรณ์แบบ

---

## หลักการทำงานของทักษะ

แต่ละทักษะจะแนะนำ Claude ผ่านกระบวนการที่มีโครงสร้าง:

1. **การค้นพบ** — ถามคำถามเฉพาะเจาะจงเกี่ยวกับสถานการณ์ของคุณ
2. **กรอบงาน** — ใช้วิธีการที่ผ่านการพิสูจน์แล้ว (PAS สำหรับ sales copy, AIDA สำหรับโฆษณา ฯลฯ)
3. **ผลลัพธ์** — ส่งมอบผลลัพธ์ที่สมบูรณ์พร้อมใช้งานด้วยรูปแบบมืออาชีพ

ทักษะไม่ใช่แค่ prompt ธรรมดา แต่เป็นระบบที่มีโครงสร้างซึ่งให้ผลลัพธ์ที่สม่ำเสมอและมีคุณภาพสูงทุกครั้ง

---

## เคล็ดลับ

- **เริ่มต้นด้วยสิ่งที่คุณต้องการวันนี้** คุณไม่จำเป็นต้องติดตั้งทักษะทั้งหมดกว่า 500 รายการพร้อมกัน เลือกหมวดหมู่ที่เกี่ยวข้องกับงานของคุณ
- **ให้ Claude ตรวจจับทักษะโดยอัตโนมัติ** คุณไม่จำเป็นต้องจำคำสั่ง slash ทั้งหมด แค่อธิบายงานของคุณตามธรรมชาติ
- **ทักษะสามารถใช้ร่วมกันได้** ใช้ทักษะหนึ่งสร้างบล็อกโพสต์ อีกทักษะนำมาดัดแปลงเป็นเนื้อหาโซเชียลมีเดีย และอีกทักษะเขียนอีเมลโปรโมทมัน
- **ปรับแต่งได้อย่างอิสระ** แต่ละทักษะเป็นไฟล์ Markdown ธรรมดา แก้ไข `SKILL.md` ใดก็ได้เพื่อปรับกระบวนการ โทน หรือรูปแบบผลลัพธ์ให้ตรงกับแบรนด์ของคุณ
